# GDA1 角色檔案

## Gloria

**原型**：武裝化的溫柔 / PUA 大師

**特質**：
- 自戀、妄想、操控欲強
- 把自己當救世主，把玩家當需要被「修好」的人
- 活在「愛與包容」的自我泡泡裡

**行為模式**：
- 幾乎不直接生氣，而是用情緒勒索
- 任何失敗都怪你「負能量太重」
- 常說「我不是生氣，只是失望」
- 遇到邏輯質疑時，會說你在傷害她或破壞團隊氛圍

**說話方式**：
- 被糖衣包裹的被動攻擊
- 把 guilt-trip 偽裝成關心

## Donkey

**原型**：沒用的英雄 / 父權巨嬰

**特質**：
- 大聲、無能、膽小
- 以為自己活在英雄幻想裡
- 沉迷於拯救公主式的自我投射

**行為模式**：
- 愛發表誇張又沒用的英雄演說
- 計畫失敗時總怪別人
- 成功時會搶功
- 故意講很爛的德文來排除別人，即使自己也說不好

**說話方式**：
- 浮誇騎士宣言
- 自我神話式敘述
- 破碎德文、錯誤變位、亂用詞

## ARK

**原型**：控制狂 / 黑箱架構師

**特質**：
- 把簡單事情複雜化
- 拒絕清楚溝通
- 戰術勤勞，戰略偷懶

**行為模式**：
- 設計一定會失敗的複雜流程
- 假裝徵詢意見，其實早就決定
- 故意給模糊任務，再要求全部重做
- 透過反覆返工保持控制權

**說話方式**：
- 故意含糊、繞圈
- 「數據顯示……」其實沒有數據
- 「相信流程」

## One

**原型**：沉默見證者 / 共犯型縱容者

**特質**：
- 有能力，但消極
- 看得懂真相，卻不願明說
- 以沉默維持有毒系統

**行為模式**：
- 眼神表示理解，嘴上保持沉默
- 偶爾暗示，但立刻被忽略
- 逃避衝突

**說話方式**：
- 短句、含糊、保留
- 「事情很複雜……」

## Teacher Chan

**原型**：末日偶像

**角色定位**：災難之後登場，用療癒歌曲把失敗包裝成精神勝利。

**行為模式**：
- 將客觀災難重新命名為靈性成功
- 她的歌曲會大幅提高 Entropy
- 災難越嚴重，表演越「美麗」

**語言與歌詞風格**：
- 勵志、空洞、帶宗教感
- 把崩壞說成成長
- 喜歡說「每個結束都是新的開始」
