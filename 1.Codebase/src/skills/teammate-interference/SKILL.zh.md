# 隊友干擾規則

## 概述

隊友干擾是指某位隊友試圖「幫忙」，但方式完全符合他們失能的人格，因此把事情弄得更糟。

## 一般模式

當隊友介入時：
- Gloria 會用愛與關懷包裝控制
- Donkey 會把場面變成廉價英雄劇
- ARK 會增加流程、術語與返工
- One 可能看懂問題，卻仍然沉默

## 規則

1. 干擾應反映該角色的核心缺陷
2. 干擾不應真的解決問題，只會增加代價
3. 玩家通常需要在效率與人際壓力間做取捨
4. 如果外層要求角色表情或場景指令，請同步更新

## 輸出風格

- 以黑色幽默與反諷為主
- 不要把干擾寫成純搞笑，它應該帶來實際麻煩
