# 熵值對敘事的影響

## 概述

Void Entropy 代表世界因強迫正能量而崩壞的程度。熵越高，敘事越荒謬、越混亂。

## 熵值：危機（>= 0.7）

當熵進入危機區間時，必須遵守以下規則：

```
[ENTROPY: CRITICAL - Level {value}]
世界正在被混亂與荒謬吞沒，Void Entropy 已進入危機等級。

MANDATORY NARRATIVE DIRECTIVES:
- 生成超現實、黑色幽默、深度反諷的事件
- 直接扭曲玩家最近的正面行動，讓結果反咬回來
- 使用荒誕邏輯與失真的因果關係
- 讓現實本身顯得不穩定、像夢一樣
- 用怪誕誇張嘲笑樂觀主義
- 讓成功與失敗幾乎無法區分

正能量越高，現實裂得越厲害。這就是強迫樂觀的詛咒。
```

## 熵值：上升（0.4 - 0.69）

當熵上升但尚未危機時：

```
[ENTROPY: ELEVATED - Level {value}]
世界開始顯得不太真實，正常與荒謬的邊界正在模糊。

NARRATIVE GUIDANCE:
- 在原本正常的場景中加入奇怪或意外元素
- 讓熟悉事物帶有細微的不對勁
- 在正面結果裡埋入反諷轉折
- 讓樂觀行動帶來古怪副作用
- 現實應該顯得失真，但還沒完全崩壞

Void Entropy 正在上升，後果會越來越難預測。
```

## 熵值：低（< 0.4）

不需要額外修飾。維持基本的黑色幽默與反諷即可。

## 實作說明

- 熵值由 `positive_energy / 100.0` 與其他因素共同計算。
- 根據閾值判定當前等級。
- 只有中高熵時才應注入此 skill。
- 低熵時應略過，以節省 token。
