# 祈禱系統：災難生成

## 概述

玩家向「飛天義大利麵神」祈禱時，願望表面上會被實現，但結果總會以更扭曲的方式造成新災難。這是遊戲諷刺強迫正能量的核心機制。

## 輸入資訊

- `{prayer_text}`：玩家祈禱內容
- `{reality_score}`：越低越容易被扭曲
- `{positive_energy}`：越高越容易盲目樂觀
- `{distortion_level}`：依現實值推導的扭曲程度

## 規則

1. 願望必須被「錯誤地實現」
2. 表面上像回應祈禱，實際上放大代價
3. 結果應帶有黑色幽默與宇宙惡意
4. 若玩家許下太正面的願望，後果通常更諷刺

## 風格

- 偽神蹟
- 荒謬因果
- 樂觀話術包裹災難
