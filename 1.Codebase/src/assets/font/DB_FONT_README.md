# Optional DB German Font

This project can load an optional DB German font at runtime:

- Expected file path: `res://1.Codebase/src/assets/font/DBNeo-Regular.otf`
- UI display name: `DB Neo`
- Language bucket: `de`

## Why the file is not included

The DB Type / DB Neo family is not bundled here because its official usage terms restrict usage to Deutsche Bahn projects.

## How to enable locally

1. Ensure you have the legal right/license to use DB Neo for your project.
2. Put the font file at:
   - `1.Codebase/src/assets/font/DBNeo-Regular.otf`
3. Launch the game. The font option will appear automatically in German font settings.
